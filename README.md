# École La Victoire - Site Web Officiel

Bienvenue sur le dépôt du site web officiel de l'**École La Victoire**, un établissement d'enseignement privé d'excellence situé à Tétouan, au Maroc.

Ce projet est une modernisation complète d'un template scolaire de haute qualité, personnalisé pour refléter l'identité visuelle et les valeurs pédagogiques de l'École La Victoire.

## 🏫 À propos de l'École La Victoire

L'École La Victoire offre un environnement d'apprentissage stimulant pour la Crèche, la Maternelle et le Primaire. Notre approche repose sur :
- **Excellence Académique** : Un programme rigoureux et innovant.
- **Trilinguisme** : Apprentissage renforcé de l'Arabe, du Français et de l'Anglais dès le plus jeune âge.
- **Formation Religieuse** : Salle de Coran dédiée et apprentissage des valeurs.
- **Soft Skills & Technologie** : Éveil à l'informatique et développement personnel.
- **Sécurité & Transport** : Installations sécurisées et service de transport scolaire professionnel.

## 🏗️ Architecture du Site (Nouvelle Version)

Le site a été entièrement restructuré pour offrir une arborescence claire et adaptée au système scolaire marocain :
- **Portails Cycles** : Landing pages spécifiques pour le `cycle-maternelle.html` et le `cycle-primaire.html`.
- **Détails des Classes** : 9 pages dédiées pour chaque niveau (de la Petite Section à la 6ème AP), connectées aux formulaires d'inscription.
- **Menu Intelligent** : Un mega-menu repensé séparant visuellement la découverte des **Cycles** et l'accès direct aux **Classes**, aussi fluide sur PC que sur Smartphone.
- **Services** : Pages dédiées à l'informatique, l'enseignement trilingue, les soft skills et le transport.

## 🛠️ Technologies Utilisées

Ce site a été développé avec des technologies modernes pour garantir fluidité et performance :
- **Frontend** : HTML5, Vanilla CSS (CSS3), Bootstrap 5.
- **Backend (Contact)** : Script PHP mailer personnalisé.
- **Animations** : GSAP, ScrollTrigger, WOW.js.
- **Interactivité** : Swiper.js (Sliders), Lenis (Smooth Scroll), Magnific Popup.
- **Icons** : FontAwesome 6 Pro.
- **Typeface** : Polices "Outfit" et "Amatic SC" via Google Fonts.

## 🚀 Fonctionnalités Clés

- **Design Premium & Responsive** : Optimisé pour tous les écrans (Desktop, Tablette, Mobile) avec un menu Hamburger unifié.
- **Navigation Fluide** : Menu intelligent avec ancres pour une expérience utilisateur simple.
- **Formulaires de Préinscription** : Systèmes d'inscription dynamiques pour envoyer les soumissions directement à `contact@groupelavictoire.com`.
- **Localisation 100% Française** : Scripts Python déployés pour garantir qu'aucune chaîne de caractères anglaise du template original ne subsiste ("Cancel Preloader", "Read More", etc.).
